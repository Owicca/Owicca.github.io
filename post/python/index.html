generators:

  • yield: the equivalent of return for a generator
  • flow:
    • when you call a function with yield the value returned is a type of iterator, a generator
    • every time you iterate through a generator(in a for-loop or calling .next()) the generator executes until it reaches an yield, returns the yield-ed value and pauses
    • if a iterator arrives at the end, it is exhausted
