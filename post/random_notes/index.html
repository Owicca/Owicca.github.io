Some algorithm:

  1. get a directory and a hash
  2. in directory there can be 0 or more children
  3. if there are more than 0 children get the hash keys of the children
  4. iterate through the keys
  5. truncate the curent key to the hash size and compare them
  6. if they are the same, if the child is a file return the file path, if the child is a directory, goroutine recursively
  7. else go to the next
  8. if no match is found, return nil

Set “offline” in lol chat:

  • RADS > Projects > lol_air_client_config_[your server here] > releases > 0.0.0.55 > deploy > lol.properties
  • xmpp_server_url
  • cmd block connection: netsh advfirewall firewall add rule name="blocklolchat" dir=out remoteip={your.region.ip} protocol=TCP action=block
  • cmd unblock connection: netsh advfirewall firewall delete rule name="blocklolchat"
