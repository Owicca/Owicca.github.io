General:

  • the public data/function members are your interface
  • put the interface first
  • methods with implementation declared in the class are generated inline at every use, giving a speed boost(no jump to common code), inlined methods do not benefit from the boost if they are larger than 4-5 lines

I/O:

  • a stream can be in four states:
    • good(): read/write was successful
    • eof(): internal file pointer is at the end of the line
    • fail(): format error level of error(you should clear() to go back to a good())
    • bad(): disk error level of error
