Trends - A trend is defined as a pattern of change.

  • sns.lineplot - Line charts are best to show trends over a period of time, and multiple lines can be used to show trends in more than one group.

Relationship - There are many different chart types that you can use to understand relationships between variables in your data.

  • sns.barplot - Bar charts are useful for comparing quantities corresponding to different groups.
  • sns.heatmap - Heatmaps can be used to find color-coded patterns in tables of numbers.
  • sns.scatterplot - Scatter plots show the relationship between two continuous variables; if color-coded, we can also show the relationship with a third categorical variable.
  • sns.regplot - Including a regression line in the scatter plot makes it easier to see any linear relationship between two variables.
  • sns.lmplot - This command is useful for drawing multiple regression lines, if the scatter plot contains multiple, color-coded groups.
  • sns.swarmplot - Categorical scatter plots show the relationship between a continuous variable and a categorical variable.

Distribution - We visualize distributions to show the possible values that we can expect to see in a variable, along with how likely they are.

  • sns.histplot - Histograms show the distribution of a single numerical variable.
  • sns.kdeplot - KDE plots (or 2D KDE plots) show an estimated, smooth distribution of a single numerical variable (or two numerical variables).
  • sns.jointplot - This command is useful for simultaneously displaying a 2D KDE plot with the corresponding KDE plots for each individual variable.

  • inferential:
    • from a sample dataset infer conclusions about the entire data set
    • why sample?:
      • collecting data from the entire population might be too time consuming
      • collecting data from the entire population might be too expensive
      • collecting data from the entire population might be impossible:
        • on a study on turtles is impossible to find every turtle, so a sample suffices
    • terms:
      • population: entire dataset
      • sample: a representative subset of the population
      • sample representation: the characteristics of the sample closely match the characteristics of the population
    • sampling methods:
      • simple random sampling: randomly select from the population using a random generator
      • sistematic random sample: set a random entrypoint and the select every n-th from there
      • stratified random sample: split the population in groups and select at random n from every group
    • sample size:
      • if the sample is large enough, all the variability in the population will pop up in the sample:
      • population size:
        • sample size is proportional to the population size
      • confidence level:
        • the confidence level you want dictates the sample size (90%, 95%, 99%)
      • margin of error:
        • the margin of error is directly proportional with the sample size, lower sample size results in a lower margin of error
    • good to know:
      • there will ALWAYS be sampling error, the sample will never be perfectly representative of the population
    • types:
      1. hypothesis test:
        • use sample to draw conclusions about the population
      2. confidence intervals:
        • taking into account sampling error, we estimate a conclusion with a confidence level (90%, 95%, 99%)
      3. regression:
        • study the relation between two variables in a population
