phpunit cli-xml equivalents:

Test Selection Options:

  • –filter Filter which tests to run
  • –testsuite Filter which testsuite to run
  • –group Only runs tests from the specified group(s)
  • –exclude-group Exclude tests from the specified group(s)
  • –list-groups List available test groups
  • –list-suites List available test suites
  • –list-tests List available tests
  • –list-tests-xml List available tests in XML format
  • –test-suffix Only search for test in files with specified suffix(es). Default: Test.php,.phpt

Test Execution Options:

  • –dont-report-useless-tests phpunit-beStrictAboutTestsThatDoNotTestAnything

  • –strict-coverage phpunit-beStrictAboutCoversAnnotation

  • –strict-global-state phpunit-beStrictAboutChangesToGlobalState

  • –disallow-test-output phpunit-beStrictAboutChangesToGlobalState

  • –disallow-resource-usage phpunit-beStrictAboutResourceUsageDuringSmallTests

  • –enforce-time-limit phpunit-enforceTimeLimit / timeoutForSmallTests / timeoutForMediumTests /timeoutForLargeTests

  • –default-time-limit phpunit-defaultTimeLimit

  • –disallow-todo-tests phpunit-beStrictAboutTodoAnnotatedTests

  • –process-isolation phpunit-processIsolation

  • –globals-backup phpunit-backupGlobals

  • –static-backup phpunit-backupStaticAttributes

  • –colors phpunit-colors

  • –columns phpunit-columns

  • –columns max phpunit-columns

  • –stderr phpunit-stderr

  • –stop-on-defect phpunit-stopOnDefect

  • –stop-on-error phpunit-stopOnError

  • –stop-on-failure phpunit-stopOnFailure

  • –stop-on-warning phpunit-stopOnWarning

  • –stop-on-risky phpunit-stopOnRisky

  • –stop-on-skipped phpunit-stopOnSkipped

  • –stop-on-incomplete phpunit-stopOnIncomplete

  • –fail-on-incomplete phpunit-failOnIncomplete

  • –fail-on-risky phpunit-failOnRisky

  • –fail-on-skipped phpunit-failOnSkipped

  • –fail-on-warning phpunit-failOnWarning

  • -v|–verbose phpunit-verbose

  • –debug Display debugging information

  • –printer phpunit-printerClass / phpunit-printerFile

  • –order-by phpunit-executionOrder

  • –random-order-seed Use a specific random seed for random order

  • –cache-result phpunit-cacheResult

  • –do-not-cache-result Do not write test results to cache file

  • Configuration Options:

  • –prepend A PHP script that is included as early as possible

  • –bootstrap bootstrap

  • -c|–configuration

  • –no-configuration

  • –extensions phpunit-extensionsDirectory

  • –no-extensions Do not load PHPUnit extensions

  • –include-path <path(s)> php-includePath

  • -d <key[=value]> php-ini

  • –cache-result-file phpunit-cacheResultFile

  • –generate-configuration Generate configuration file with suggested settings

  • –migrate-configuration Migrate configuration file to current format

  • convertDeprecationsToExceptions
  • convertErrorsToExceptions

  • convertWarningsToExceptions

only xml:

  • php-const: define a value
  • php-var: set a value in $GLOBALS
  • php-env: set a value in $_ENV
  • php-get: set a value in $_GET
  • php-post: set a value in $_POST
  • php-cookie: set a value in $_COOKIE
  • php-server: set a value in $_SERVER
  • php-files: set a value in $_FILES
  • php-request: set a value in $_REQUEST
