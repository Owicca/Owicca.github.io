  • #2 => Rules for addition:
    • commutativity:
      • a, b E integers
      • a + b = b + a
    • asociativity:
      • a, b, c E integers
      • (a + b) + c = a + (b + c)
    • N3 => if a + b = 0, then b = -a and a = -b:
      • proof 1: a + b = 0 /+(-a) -a + a + b = -a + 0 b = -a => -(-a) = -b = a
      • proof 2: a + b = 0 /+(-b) a + b - b = -b + 0 a = -b => -(-b) = -a = b
    • N4 => a = -(-a):
      • proof 1: a + (-a) = 0
    • N5 => -(a + b) = -a - b
      • proof 1: if a, b are integers then x = -y and y = -x, means x + y = 0 (a + b) + (-a - b) = 0 (a + b) + (-a - b) = a + b - a - b(commutativity) = a - a + b - b(associativity) = 0
    • if a, b are positive integers; a + b = positive integer
    • if a, b are negative integers; a + b = negative integer
      • proof 1: m, n positive a = -n, b = -m a + b = -n - m = -(n + m) a + b is negative because n + m is positive
      • a + b = c
      • a = c - b
      • b = c - a
    • cancellation law with integers: if a + b = a + c, then b = c
  • #3 => Rules for multiplication:
