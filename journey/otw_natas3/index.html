  1. Google indexes websites by following links from external sites that point to your website(backlinks).
    Spiders(the google bots) are well-behaved bots, that first request robots.txt file from your domain(domain.com/robots.txt).
    The purpose of the file is to control the requests from well-behaved bots.
    The file should follow a well defined format that instructs the bot what endpoints are public and what should not be crawled(unimportant pages, duplicate pages, etc).
