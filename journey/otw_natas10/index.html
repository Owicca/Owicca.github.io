steps:

  1. exactly the same setup as natas9, but this time there is a weak filtering on user input /[;|&]/
  2. we will try to hunt for /etc/natas_webpass/natas11 at the same time circumventing the filter, the filter denies the use of:
    • ;: multiple commands on the same line
    • |: piping
    • &: sending a process to the background
  3. if we are stuck with the current grep, we can exploit the fact that grep can search through multiple files
  4. by aiming for a grep of the entire contents of multiple files(grep -i '.*' f1 f2 f3 fn), we try '.*' /etc/natas_webpass/natas11 (take note of the space at the end of the input) and get the flag
